# -A-Web-Based-Smart-Library-Management-System
Library Track: A Web-Based Smart Library Management System is designed to improve the efficiency of library operations by replacing traditional manual record keeping with a digital platform. 
The proposed system is  primarily admin-focused, where administrator manage books, register students and handle book issue and return  processes through a centralized web interface.technologies: HTML, CSS, Bootstrap 5, PHP, MySQL, XAMPP.
