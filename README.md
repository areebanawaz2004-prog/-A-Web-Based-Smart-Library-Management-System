# -A-Web-Based-Smart-Library-Management-System
The management of the library system in educational institutions is currently done manually. In order to search for, borrow or return books, the users should visit the library. This way of doing things is time consuming and error-prone and cannot be described as being effective. 
In order to resolve the above challenges, our suggestion is to create a Web-Based Library Management System (LMS). This LMS will have automated all necessary functions performed by the library such as user registration, searching, borrowing, returning, reserving, computation of fines and reports.
HTML, Bootstrap, PHP and MySQL technologies will be used in creating this LMS.
